2. Объясните, в чем состоят различия между файлом исходного кода, файлом объектного кода и исполняемым файлом?

	Файл исходного кода - не скомпелированный файл, который содержит в себе код программы на языке программирования.
	Файл объектного кода - Созданный компилятором файл из исходного, имеющийрасширение .о.
	Исполняемый файл - Файл, запускающий уже скомпилированную программу.
